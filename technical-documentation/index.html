Looping in Python

What Are Loops?

Loops are a way to simplify and shorten your code. They are great for iteration and repeating simple calculations.

For example, say you had the following code.

    i = 0 
    i += 1 
    print(i) 
    >>1 
    i += 1 
    print(i) 
    >>2

This simple program declares a variable i and initializes it equal to 0. The program then increments i by 1 and then prints the value of i to the screen. It completes this process a total of two times. All in all a total of 5 lines of code. Not bad. But what if we wanted to do this 10 times, or 100 times? What if our program needed to do this 1,000 times?

This is where loops come in to play.

For Loop

The first type of loop we will discuss is the For Loop. The for loop has a simple syntax that is easy to understand.

    for each in (any iterable): 
    do something

The for loop starts with the reserved python keyword "for" and next declares a variable, in our example "each". This variable can be called anything, but to keep your code easy to follow it's best to name it the singular version of your iterable's name. For example if you had a list variable with the name "items" most pythoner's would do 

    for item in items:
      do something...

For loops can be used to iterate over any type of iterable object. Any of the following can be used with a for loop

  • Lists
  • Dictionaries
  • Sets
  • Tuples
  • Strings

Next, the body of the for loop is where you put what you want to happen for each round of the for loop. Let's look at our original example but this time lets have it go for 10 times

    for i in range(10):
        i += 1
        print(i)

    >>1
    >>2
    ..
    ..
    >>9
    >>10

Look at that glorious for loop in action. Only took three lines what would have taken us 20. What if we wanted to make it go 100 times? Just change the range() number. for i in range(100):

1,000? 10,000? 100,000 times?? You guessed. We just change the amount passed into the range function, but the remainder of the for loop stays the same. This is the power of the for loop.

While Loop

For loops are great for when we know exactly how many times we want to iterate through the loop. But what if we didn't? What if we wanted the loop to execute until a certain condition was met but we didn't know exactly when that would happen?

One way to do this is to add an if statement to our for loop body and if it evaluates to true, execute a "break" statement that will exit the loop

    sum = 0
    for num in nums:
      if num == 2:
        break
      sum += num

The above for loop will iterate over through the iterable named nums and checks to see if each num is equal to 2. If the current num is equal to 2, the break statement is excecuted and the for loop is exited. Otherwise, the num gets added to the variable sum and the loop continues iterating over nums

This certainly gets the job done. But there is another way. Enter the while loop.

    running = True
    while running:
        print("Enter a letter:",
             "(press q to quit)")
        ans = input()
        if ans == "q":
            running = False
        else:
            do something

The code above has a variable "running" initialized to the boolean value of True. The while loop condition checks to see if running == True. If so it prompts the user for input and saves the response to the variable "ans". If the user enters anything other than "q", the else statement is entered and the loop repeats until the user enters "q". Once the user presses "q" the if statement is entered and running is changed to False. This will fail the while loop condition and then the loop is exited. This type of scenario is most common in games and other programs where the program depend on user input to exit.

Nested Loops

Sometimes there are situations where you may find it useful to have one loop nested inside another loop. For example, say you had a two dimensional list, that is a list that contains other lists

    nums = [[0,1,2,3],[4,5,6,7]]

When you need to iterate over each individual item in the list you can used a nested for loop

    nums = [[0,1,2,3],[4,5,6,7]]
    for i in range(len(nums)):
        for j in range(len(nums[i])):
            print(nums[i][j])
            
    >>0
    >>1
    ..
    ..
    >>6
    >>7

You can also nest while loops.

    var1 = True
    var2 = True
    
    while var1:
        while var2:
            if something == this:
                var2 = False
            else:
                do something...
        if something == this:
            var1 = False
        else:
            do something...

In some cases, you can even nest a for loop inside a while loop and vice versa

    for i in range(10):
      while i < 5:
          print(i)
          i += 1
    
    i = 0
    while i < 10:
        for j in range(5):
            i += j

While these setups can be useful, caution is needed as they can be tricky to debug and there usually is a simpler way to accomplish the same task

Infinite Loop

The last topic of discussion pertains only to the while loop. When constructing your while loop, it is critical that your exit condition is actually possible to be executed. If not, you will end up with the dreaded Infinite Loop For example.

    i = 0
    while i < 10:
      i -= 1
      print(i)
      
    >> -1
    >> -2
    ..
    ..
    >> -infinity

The while loop above will never exit as the conditionan i less than 10, will always be true since in the body of the loop, i is decremented instead of incremented so i will never be greater than 10 and the loop just goes on and on and on and on...
